# Installation

The Command Block Staff Addon is an **add-on** — it needs the base **AvM Staff Mod** installed alongside it.

## Required mods

=== "Fabric"

    - [AvM Staff Mod](https://modrinth.com/mod/avm-staff) (the base mod)
    - [Fabric API](https://modrinth.com/mod/fabric-api)
    - [Fabric Language Kotlin](https://modrinth.com/mod/fabric-language-kotlin)
    - [Architectury API](https://modrinth.com/mod/architectury-api)
    - [GeckoLib](https://modrinth.com/mod/geckolib) *(for the ride animation)*

=== "NeoForge"

    - [AvM Staff Mod](https://modrinth.com/mod/avm-staff) (the base mod)
    - [Architectury API](https://modrinth.com/mod/architectury-api)
    - [Kotlin for Forge](https://modrinth.com/mod/kotlin-for-forge)
    - [GeckoLib](https://modrinth.com/mod/geckolib) *(for the ride animation)*

**Minecraft 1.21.** Optional: [Mod Menu](https://modrinth.com/mod/modmenu) + optionally
[Cloth Config](https://modrinth.com/mod/cloth-config) for a nicer settings screen.

## Steps

1. Install your loader (Fabric or NeoForge) for **Minecraft 1.21**.
2. Drop the required mods above **plus** the Command Block Staff Addon jar into your `mods` folder.
3. Launch the game.

## Getting the staff

1. Craft / obtain a **Royal Staff** or **Scepter of Friendship** from the base AvM Staff Mod.
2. Put a **Command Block** into the staff (the base mod's insert key, default **R**).
3. Hold the staff — you're ready. Press **Z + scroll** to pick an ability.

!!! warning "Items must be allowed in the staff"
    The command block is enabled in the staff out of the box. If you ever see *"cannot be used in staff,"*
    the item isn't in that staff's allow-list.
