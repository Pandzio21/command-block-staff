# Projectile (Ability 4)

Launch projectiles from the staff tip toward where you aim. Open the launcher with **V** to pick a type or
manage custom ones.

## Modes

Cycle with ++left-alt++:

- **Arrow Cannon** *(Normal)* — **hold** right-click to spam your selected projectile (arrows, fireballs,
  TNT, food, or a custom type). Arrows rain on the entity you're aiming at.
- **Advanced control** — hold to guide a projectile.
- **Control Projectiles** — a defensive pool. **Spent arrows on the ground near you are collected** into a
  fan behind your back. **Left-click loads** one to the staff tip; **right-click / hold fires** it at your
  aim. Scroll picks the type (Random / Arrow / Fireball / TNT / Food).

## Controls

| Input | Action |
|-------|--------|
| Right-click | Fire (hold to spam in Arrow Cannon) |
| Left-click | Quick projectile chooser — or **load** in Control mode |
| {{ "`" }} + scroll | Throw distance |
| ++v++ | Open the launcher / manage custom projectiles |

## Custom projectiles

In the launcher → **✦ Custom Projectiles…** you can add any entity by its ID, name it, and fire it like a
built-in type. They also show up in the quick chooser.

!!! note "Control mode in survival"
    By default, spawning a controlled **arrow** in survival uses one arrow from your inventory (toggle this
    with the **Control: consume arrows** server setting). It only collects *grounded* arrows — it won't
    snatch a skeleton's shot out of the air, so it isn't a shield.
