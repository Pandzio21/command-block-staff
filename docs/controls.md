# Controls

Everything is rebindable under **Options → Controls → Command Block Staff Addon**. Defaults:

| Key | Action |
|-----|--------|
| ++z++ + scroll | Change ability (1–6) |
| ++left-alt++ | Change the current ability's mode |
| {{ "`" }} + scroll | Adjust the current ability's setting |
| ++v++ | Open the current ability's menu |
| ++f4++ | Clear electrified blocks |
| ++u++ | Undo the last command-ability action |

Mouse depends on the ability — right-click is the main action, left-click the secondary/quick-chooser, and
**Shift + click** does mode extras. Exact bindings are on each ability's page. When a mode has a Shift
action, it prints a one-line hint at the top of the screen.
