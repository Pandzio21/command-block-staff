# Controls

The addon has its **own keybind category** — *"Command Block Staff Addon"* — in
**Options → Controls**, so you can rebind anything. Defaults below.

## Core keys

| Key | Action |
|-----|--------|
| ++z++ + scroll | **Change ability** (1–6) |
| ++left-alt++ | **Change ability mode** (cycles the current ability's sub-mode) |
| {{ "`" }} + scroll | **Adjust setting** (beam distance, dome size, projectile range, etc.) |
| ++v++ | **Open ability settings** (the menu for the current ability) |
| ++f4++ | **Clear electrified** blocks |
| ++u++ | **Undo** the last command-ability action |

## Mouse

| Button | Action |
|--------|--------|
| **Right-click** | The ability's main action (grab, electrify, place, shoot, …) |
| **Left-click** | The ability's secondary action / quick-chooser (varies per ability) |
| **Shift + click** | Mode-specific extras (un-electrify, push a wall, target an entity, …) |

Each ability page lists exactly what right-click, left-click and shift do for that ability.

## Riding

| Key | Action |
|-----|--------|
| Ride keybind *(set in Controls)* | Mount / dismount the staff to fly |

See **[Ride the Staff](ride.md)**.

!!! note "On-screen hints"
    Many modes print a one-line hint at the top of the screen telling you what Shift + click does right
    now — watch for it when you switch modes.
